Вариант 11: Очередь (1)
Задание:
Реализуйте очередь с помощью связного списка
Реализуйте очередь с помощью массива
Сравните полученные реализации, в чем преимущества и недостатки каждой?

Сравнение реализаций:
Очередь на связном списке:

 Преимущества: Динамический размер, нет ограничения по емкости

 Недостатки: Большие накладные расходы памяти, медленный доступ

Очередь на массиве:

 Преимущества: Лучшая производительность кэша, меньше накладных расходов памяти

 Недостатки: Фиксированная емкость, необходимость перераспределения

Тесты покрывают различные сценарии, включая граничные случаи, операции с пустыми очередями, переполнение и циклическое поведение.
